fullPage.js plugin for WordPress - Create Beautiful Fullscreen Scrolling Websites.

A simple and easy to use plugin to create fullscreen scrolling websites (also known as single page websites or onepage sites). It allows the creation of fullscreen scrolling websites, as well as adding some landscape sliders inside the sections of the site.

== Description ==


fullPage.js plugin for WordPress - Create Beautiful Fullscreen Scrolling Websites.

The demo: http://fullpage.lamblue.com/

A simple and easy to use plugin to create fullscreen scrolling websites (also known as single page websites or onepage sites). It allows the creation of fullscreen scrolling websites, as well as adding some landscape sliders inside the sections of the site.


Features List:
- Scroll Bar Enable
- Normal Scrolling
- Without anchor links (same URL)
- Vertical navigation dots
- Loop
- Full Background
- Reponsive

== Installation ==

The documentation: http://lamblue.com/documentation/fullpage/

How to install the plugin and get it working?

1. Upload the plugin files to the `/wp-content/plugins/wp_fullpage` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Sections->All sections screen to management sections
4. Use the Sections->Add section screen to create section
5. Use the fullPage->All fullPage screen to management fullPage shortcodes
6. Use the Sections->Add fullPage screen to create fullPage shortcode
